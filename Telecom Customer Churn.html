Lots of machine learning algorithms require numerical input data, so we need to represent categorical columns in a numerical column

In order to encode this data, we could map each value to a number. e.g. gender column's value can be represented as male:0, and female:1

This process is known as label encoding, and sklearn conveniently will do this for us using LabelEncoder

from sklearn.feature_selection import RFECV

#RFECV is Recursive feature elimination with cross-validation to select features
#Create the RFE object and compute a cross-validated score
#The "accuracy" scoring is proportional to the number of correct classifications

rfecv = RFECV(estimator=LogisticRegression(max_iter=1000), cv=10)   #To fix Convergence warning specify max_iter in the LogisticRegression to a higer value

rfecv.fit(x, y)

print('Optimal number of features: %d' % rfecv.n_features_)
print('Selected features: %s' % list(x.columns[rfecv.support_]))
